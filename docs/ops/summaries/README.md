# Summaries (Compressed History)

Summaries compress multiple worklog entries into a small digest.

Path pattern:

`docs/ops/summaries/YYYY-WW.md`

Use `WEEKLY_TEMPLATE.md` for weekly rollups.

Rules:

- Link back to source worklogs/issues/PRs
- Focus on decisions, risk, and operational impact
- Keep concise and prune repetitive detail
